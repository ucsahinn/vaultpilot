## Qué Es Público

Este repositorio contiene notas de release, screenshots sanitizados, documentación de operador, artículos de knowledge base, reglas de soporte y políticas públicas. Los instaladores y ZIPs pertenecen a GitHub Releases, no al árbol Git.

La portada canónica en inglés sigue en [README.md](README.md).

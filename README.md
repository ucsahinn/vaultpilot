# PassMan Releases

Public release artifacts and signed update manifests for PassMan.

This repository intentionally contains only customer-facing download assets. The PassMan product source, build system, signing workflow, and internal development history remain in the private `ucsahinn/passman` repository.

## Latest Update Manifest

`https://github.com/ucsahinn/passman-releases/releases/latest/download/passman-update.json`

PassMan verifies the signed manifest and pinned SHA-256 checksums before exposing update packages.

- `PassMan-1.0.2-x64.msi` - PassMan Server 1.0.2 Windows installer
- `passman-chromium-extension.zip` - Chromium extension 3.1.0 fallback package
- `passman-update.json` - signed update manifest
